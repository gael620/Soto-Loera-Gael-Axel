# Soto-Loera-Gael-Axel
Tienda escolar desarrollada como parte de un proyecto academico. Permite visualizar productos, filtrarlos y mostrarlos usando JavaScript y archivos JSON.

Descripción general
Este proyecto es una página web que simula una tienda escolar. Los productos se cargan desde tres archivos JSON, y el comportamiento dinámico se maneja
mediante un archivo JavaScript. La estructura visual está definida en un archivo HTML y los estilos en un archivo CSS.
El objetivo del proyecto es demostrar el uso de:
- Lectura de datos desde JSON
- Manipulación del DOM con JavaScript
- Diseño web con HTML y CSS
- Publicación del proyecto usando GitHub Pages

Estructura del proyecto
/tienda-escolar
│── index.html        → Página principal
│── css               → style.css        → Estilos de la página
│── data              → libretas.json    → Lista de productos de libretas
│                     → libros.json      → Lista de productos de libros
│                     → utiles.json      → Lista de productos de utiles    
│── js                → app.js           → Lógica de la tienda (cargar productos, eventos, etc.)

Funcionalidades principales
- Cargar productos desde archivos JSON
- Mostrar productos en la página
- Filtrar y categorizar productos
- Mostrar precios y descripciones
- Diseño 
- Interacción básica con botones o eventos

Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
- JSON
- GitHub Pages
